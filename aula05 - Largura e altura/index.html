Largura e Altura

Largura = width, Altura = height

classe w é usada para definir o width. podendo usar as configurações do espaçamento(w-1, w-2,w-3....w-96), porcentagem (1/2, 1/3, 2/4, 1/5, 1/6, 1/12...11/12), w-full(100%), w-screen(100vh), w-min (minimo), w-max(máximo). o mesmo vale para o Height.
